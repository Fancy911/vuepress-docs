## `react-router-dom`是什么

1. `react`的一个插件库。
2. 专门用来实现一个`SPA应用`。
3. 基于`react`的项目基本都会用到此库。

## 内置组件

1.	`<BrowserRouter>`
2.	`<HashRouter>`
3.	`<Route>`
4.	`<Redirect>`
5.	`<Link>`
6.	`<NavLink>`
7.	`<Switch>`

## 其它API

1.	history对象
2.	match对象
3.	withRouter函数
