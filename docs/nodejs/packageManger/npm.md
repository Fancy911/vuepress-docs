## 2.npm的基本使用

### 初始化

创建一个空目录，然后**以此目录作为工作目录**启动命令行工具 ，执行`npm init`

- `npm init`命令的作用是将文件夹初始化为一个『包』，**交互式**创建`package.json`文件。
- `package.json`是包的配置文件，每个包都必须要有`package.json`
